# Data Sources & Methodology

## 🔍 Data Quality Classifications

All data in Public Money Mirror is classified into three categories:

### ✅ **Official**
- Source: Government publications, federal databases
- Accuracy: Directly from primary sources
- Verification: Can be independently verified with cited sources

### ⚠️ **Estimated**
- Source: Derived from official data using standardized methods
- Accuracy: Based on reasonable assumptions and calculations
- Verification: Methodology is transparent and documented

### 🎭 **Mock/Illustrative**
- Source: Created for demonstration purposes
- Accuracy: Representative but not actual data
- Verification: Clearly marked as example/demonstration data

## 📏 Calculation Methods

### Per-Citizen Metrics
- **Population**: 83,200,000 (German population estimate)
- **Average Household Size**: 2.1 persons
- **Formula**: `per_citizen = total_amount / population`

### Budget Percentages
- **Formula**: `percentage = (category_amount / total_budget) * 100`
- **Rounding**: 2 decimal places

### Historical Growth
- **Formula**: `growth_pct = ((final_amount - initial_amount) / initial_amount) * 100`
