# Analytics Notebooks

This directory contains Jupyter notebooks for exploratory data analysis and model development.

## Setup

```bash
pip install jupyter notebook
jupyter notebook
```

## Notebooks

- `case_ranking_exploration.ipynb` - Explore case ranking features and model performance
- `anomaly_detection_tuning.ipynb` - Tune anomaly detection thresholds
- `benchmark_analysis.ipynb` - Analyze unit price benchmarks

## Guidelines

- Keep notebooks focused on specific analyses
- Document assumptions and methodology
- Export useful code to Python modules in `../features/` or `../models/`
