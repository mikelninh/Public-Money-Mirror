## RESPONSIVE BREAKPOINTS

### Mobile (320px - 767px)
- Single column layout
- Reduced typography (64px → 36px hero)
- Reduced spacing (96px → 48px sections)
- Stacked buttons
- Hamburger navigation

### Tablet (768px - 1023px)
- 2-column grid where appropriate
- Moderate typography reduction
- Side-by-side buttons (if space allows)

### Desktop (1024px+)
- Full 12-column grid
- Maximum width: 1280px container
- All spacing and typography at full scale

## ANIMATION TIMELINE EXAMPLE

### Page Load Sequence

```
0ms    →  Page loads, elements invisible
150ms  →  Hero title fades in (opacity 0 → 1, translateY 20px → 0)
300ms  →  Hero subtitle fades in
450ms  →  Visual element (transparent circle) fades in
600ms  →  Body text fades in
750ms  →  Buttons fade in with slight delay
900ms  →  Background elements (if any) complete fade-in
```

**Total**: ~1 second for full hero reveal

### Hover Interactions

```
0ms    →  Mouse enters element
150ms  →  Transform translateY(-4px) complete
        →  Shadow increases
        →  Border color changes (if applicable)
        →  Opacity adjusts (if applicable)
```

## ICON SPECIFICATIONS

### Size Scale
- Small: 16px (inline with text)
- Medium: 24px (buttons, cards)
- Large: 48px (feature cards)
- Hero: 64px+ (decorative)

### Style
- Line weight: 1.5-2px
- Rounded corners: 2px radius
- Color: #2D8659 (primary), #4A90E2 (secondary), #D4AF37 (accent)
- Background: Transparent or subtle circle

### Icon Set Recommendations
- Feather Icons (minimal, elegant)
- Heroicons (geometric, clean)
- Custom (geometric + organic blend)
